## 优化

- 将 Google OpenAI 兼容模式的自定义模型 ID 从“按供应商共享”调整为“按具体模型分别配置”，避免 Gemini 与 Imagen 互相串配置
- 设置页拆分 Google 两个模型的独立自定义 ID 输入（Gemini 2.0 Flash、Imagen 3），并在读取/保存时自动清理旧版 `google` 共享键

## 修复

- 新增 Google OpenAI 兼容模式下自定义模型 ID 的前置校验，对明显非图片生成模型给出可读错误，避免上游 500 难定位
- 兼容迁移旧配置：自动将历史 `google` 自定义 ID 映射到对应的 Gemini 或 Imagen 模型键
